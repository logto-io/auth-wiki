## ロール (Role)

ロール (Role) は、<Ref slug="rbac" /> のコアコンポーネントの一つです。これは、ユーザーに割り当てることができる権限のコンテナとして機能し、ユーザーとそのアクセス権の間の仲介役を果たします。

```mermaid
  graph LR
    Users
    -->|assigned to| Roles
    -->|including| Permissions
```

典型的なロールの構造は、名前と一連の権限を含みます:

```typescript
const role = {
  name: 'order_admin',
  permissions: [
    'read:orders',   // 注文詳細の表示
    'write:orders',  // 注文の編集
    'read:products'  // 製品の表示
  ]
}
```

> [!Note]
> ロールは主に権限管理に使用されます。アクセス制御の実装には、ロールではなく権限を直接確認することをお勧めします。詳細は <Ref slug="rbac" /> を参照してください。
