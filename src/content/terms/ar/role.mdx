## دور (Role)

الدور هو أحد المكونات الأساسية في <Ref slug="rbac" />. يعمل كحاوية للأذونات التي يمكن تعيينها للمستخدمين، ويعمل كوسيط بين المستخدمين وحقوق الوصول الخاصة بهم.

```mermaid
  graph LR
    المستخدمون
    -->|معين إلى| الأدوار
    -->|بما في ذلك| الأذونات
```

تحتوي بنية الدور النموذجية على اسم ومجموعة من الأذونات:

```typescript
const role = {
  name: 'order_admin',
  permissions: [
    'read:orders',   // عرض تفاصيل الطلب
    'write:orders',  // تعديل الطلبات
    'read:products'  // عرض المنتجات
  ]
}
```

> [!Note]
> تُستخدم الأدوار بشكل أساسي لإدارة الأذونات. لتنفيذ التحكم في الوصول، يُوصى بفحص الأذونات مباشرة بدلاً من الأدوار. راجع <Ref slug="rbac" /> لمزيد من المعلومات.
